# Firmware Releases

This directory contains versioned firmware releases for the OPC UA Server for Kincony KC868-A16V3.

## Latest Release
**v1.0.0** - Initial release (December 11, 2024)

## Directory Structure
```
firmware/
├── v1.0.0/
│   ├── opcua-kincony-a16v3.bin
│   ├── bootloader.bin
│   ├── partition-table.bin
│   └── README.md
└── README.md
```

## How to Use
1. Navigate to the desired version directory
2. Follow the flashing instructions in the version's README.md
3. Use the checksums to verify file integrity
